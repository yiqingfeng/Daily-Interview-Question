## 最长回文子串

### 题目

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

**示例 1：**
```
输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。
```

**示例 2：**
```
输入: "cbbd"
输出: "bb"
```

### 解答 JavaScript

#### 我的暴力法，枚举

```javascript
var longestPalindrome = function(s) {
    const len = s.length;
    if (len < 2) {
        return s;
    }
    if (len === 2) {
        return s[0] === s[1] ? s : s[0];
    }

    let maxStr = '';
    for(let i = 0; i < len; i++) {
        // 字符为奇数 aba 的逻辑
        let curStr = s[i];
        for (let j = i - 1; j >= 0; j--) {
            let y = 2 * i - j;
            if (y === len) break;
            if (s[j] === s[y]) {
                curStr = s[j] + curStr + s[y];
                continue;
            }
            break;
        }
        if (maxStr.length < curStr.length) {
            maxStr = curStr;
        }
        // 字符为偶数 abba
        if ((i+1) < len && s[i] === s[i+1]) {
            curStr = s[i] + s[i+1];
            for (let j = i - 1; j >= 0; j--) {
                let y = 2 * i - j + 1;
                if (y === len) break;
                if (s[j] === s[y]) {
                    curStr = s[j] + curStr + s[y];
                    continue;
                }
                break;
            }
        }
        if (maxStr.length < curStr.length) {
            maxStr = curStr;
        }
    }
    return maxStr;
};
```
